本人在大三课余时间基于鸿蒙移动应用开发课堂内容和华为开发者官网，开发的音乐播放器app
DevEcoStudio版本为5.0.5
API Version 15
仿真设备版本：HuaWei-Phone HarmonyOS 5.0.5（17）
应用内获取歌曲的api使用的是QQ音乐的api，
整体播放器的功能方面已经开发完毕，ui及动画还有关于服务卡片的内容正在完善中...
